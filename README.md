> Harrison.AI API Project

## Pre-requisites

Prior to start the project locally, docker must be installed in order to start the local mysql database server.

## Install all dependencies

```
yarn install
```

## Start local Mysql Database Server

```
yarn start:db
```

## Build

```
yarn build
```

## Start

```
yarn start
```

## Test

```
yarn test
```

## Linting

```
yarn lint
```

## Fix Linting

```
yarn format
```
